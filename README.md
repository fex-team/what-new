

## 特点

* 面向实际应用，有实际使用的例子，并介绍可以在哪些场景中使用
* 指出问题，规范及相关 PR 文章往往只提到好的一面，这里将毫无保留地吐槽它不好的一面
* 丰富的相关链接，方便学习这个技术相关的细节知识

## 主要关注点

* 背景
* 浏览器支持情况
* 应用领域
* 替代方法
* 存在的问题
* 实现原理
